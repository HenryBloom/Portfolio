# Henry Bloomfield - Portfolio & CV

Welcome to my professional portfolio and CV website! This is a clean, modern, and responsive single-page CV built with HTML and CSS.

## Features

- 📱 **Fully Responsive** - Looks great on desktop, tablet, and mobile devices
- 🎨 **Beautiful Design** - Green-themed color scheme with smooth animations
- ⚡ **Fast & Lightweight** - Pure HTML and CSS, no JavaScript dependencies
- 🔗 **Navigation** - Smooth scrolling between sections
- 📄 **Organized Sections**:
  - Overview / About
  - Education
  - Work Experience
  - Projects
  - Skills

## Sections

### Overview
A brief introduction about me and my professional interests.

### Education
Information about my academic background and relevant coursework.

### Work Experience
Details of my professional roles, responsibilities, and achievements.

### Projects
Showcase of my key projects with technologies used and descriptions.

### Skills
Organized by categories: Frontend, Backend, Tools & Platforms, and Soft Skills.
